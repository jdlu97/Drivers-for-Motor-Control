Detailed Description

A driver for working with a quadrature incremental encoder.

Encapsulates functionality of encoder attached to the motor. This class sets up pins to control the encoder and methods to read and zero (reset) the angular position of the motor in units of "ticks". Position reading is based on an algorithm that accounts for overflow and underflow.
